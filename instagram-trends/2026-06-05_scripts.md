### Script 1: One operator, every platform

**Topic:** Running an entire content department as a single operator inside Claude Cowork.
**Pillar:** Future-Proofing
**Hook Type:** Founder Mirror

**Source video:** https://www.instagram.com/p/DXedV0Ykmnu/ — @sabrina_ramonov

**On-screen hook text:** "IT ALL ROUTES THROUGH YOU"
**Spoken hook:** "Your content dies the second you get busy — because every post still has to run through you."
**Hold line:** "You're not short on ideas. You're short on a system that keeps going when you step away."

**Full script:**
"Your content dies the second you get busy, because every post still has to run through you. It's not that you're out of ideas — it's that nothing moves the second you step away. We do this for clients all the time and it crushes."

- 5–10s: Flow diagram — `/post → platform → topic → script → calendar`.

**Why this hook should perform:**
It mirrors the exact bottleneck every solo operator feels — growth capped by their own hours — instead of leading with the tool. The "system that runs without you" curiosity gap pulls the watch, and "approve, don't operate" filters for people running a real content engine, not hobbyists chasing a free prompt.
